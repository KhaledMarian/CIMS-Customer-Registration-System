This repository contains a Java-based Customer Registration System developed as part of a Customer Information Management System (CIMS) project for Zain Jordan.

## Project Overview
The system allows registering new customers, validating input data, preventing duplicate records, sorting customer data, and searching using efficient algorithms.

## Features
- Customer registration with input validation
- Prevention of duplicate customer IDs
- Graphical User Interface (GUI) using Java Swing
- Data storage using ArrayList
- Merge Sort for sorting customer records
- Jump Search for efficient searching

## Technologies Used
- Java
- NetBeans IDE
- Object-Oriented Programming (OOP)
- Event-driven programming

## How to Run the Project
1. Open NetBeans IDE.
2. Select **Open Project**.
3. Choose the project folder.
4. Run the project from NetBeans.
